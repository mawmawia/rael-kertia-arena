# Rael Kertia Arena 🎮

Fair skill game bot for Telegram. Reaction clicker with transparent RNG + skill revives. No pay2win.

**Play Now:** @YourBotUsername → /start

**Why this vs Royal Match/Monopoly GO:**
- **Fair**: Adaptive difficulty + transparent seed. Same seed = same level for everyone
- **No Paywall**: Skill Revive = watch ad OR hit 3 targets. Never forced to pay
- **Social**: /duel @friend + /top group leaderboard 
- **Monetize**: Monetag interstitial + rewarded ads + Telegram Stars IAP for Premium

**Stack**
Telegram Bot API + Vercel Serverless + WebApp + Monetag

**Setup - 5 minutes**
1. BotFather → /newbot → get `BOT_TOKEN`
2. Push to GitHub → Import to Vercel → Add `BOT_TOKEN` env var
3. Set webhook: `https://api.telegram.org/botTOKEN/setWebhook?url=YOUR_VERCEL_URL/api/bot`
4. Monetag.com → Create Rewarded + Interstitial zones → paste IDs in `public/index.html`
